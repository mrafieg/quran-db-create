## Jalan Surah Quran SQL

Go Script to preapre query for creating jalan surah app quran Database 

### run with:
```
go run main.go
```

### sql run order
1. quran-surah.sql
2. quran-text.sql
3. surah-info.sql
4. ayah-info.sql
5. ayah-word-count.sql
